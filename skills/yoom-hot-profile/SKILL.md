# Hot Profile Yoom 연계 스킬

Hot Profile와 OpenClaw 연결을 위한 스킬입니다.

## 서비스 정보
- **서비스명**: Hot Profile
- **카테고리**: セールス (Sales)
- **Yoom URL**: https://lp.yoom.fun/apps/hot-profile

## 연계 정보
- **연계 방식**: API
- **인증 방식**: OAuth
- **API 액션 수**: 31個
- **트리거 수**: 13個
- **템플릿 수**: 6個

## 구현된 API 액션
1. 명사의 사용자 정의 필드 업데이트 (名刺のカスタム項目を更新)
2. 보고 관리 업데이트 (報告管理を更新)
3. 리드 검색 (リードを検索)
4. 명사 정보 등록 (名刺情報を登録)
5. 상담 업데이트 (商談を更新)
6. 상담의 사용자 정의 필드 업데이트 (商談のカスタム項目を更新)
7. 상품 마스터 검색 (商品マスターを検索)
8. 테스크 검색 (タスクを検索)
9. 명사 정보 업데이트 (名刺情報を更新)
10. 리드 업데이트 (リードを更新)
11. 회사 등록 (会社を登録)
12. 회사 검색 (会社を検索)
13. 리드의 사용자 정의 필드 업데이트 (リードのカスタム項目を更新)
14. 보고 관리의 사용자 정의 필드 업데이트 (報告管理のカスタム項目を更新)
15. 리드 정보 가져오기 (リード情報を取得)
16. 상담의 필드 정보 가져오기 (商談のフィールド情報を取得)
17. 상담 검색 (商談を検索)
18. 상품 마스터 업데이트 (商品マスターを更新)
19. 보고 관리 등록 (報告管理を登録)
20. 회사의 필드 정보 가져오기 (会社のフィールド情報を取得)
21. 상품 마스터 등록 (商品マスターを登録)
22. 보고 관리 검색 (報告管理を検索)
23. 리드의 필드 정보 가져오기 (リードのフィールド情報を取得)
24. 리드 등록 (リードを登録)
25. 회사 정보 가져오기 (会社情報を取得)
26. 회사의 사용자 정의 필드 업데이트 (会社のカスタム項目を更新)
27. 보고 관리의 필드 정보 가져오기 (報告管理のフィールド情報を取得)
28. 회사 업데이트 (会社を更新)
29. 명사의 필드 정보 가져오기 (名刺のフィールド情報を取得)
30. 상담 등록 (商談を登録)
31. 명사 검색 (名刺を検索)

## 구현된 트리거
1. 명사가 등록되면 (名刺が登録されたら)
2. 테스크가 업데이트되면 (タスクが更新されたら)
3. 리드가 업데이트되면 (リードが更新されたら)
4. 회사가 업데이트되면 (会社が更新されたら)
5. 테스크가 생성되면 (タスクが作成されたら)
6. 보고 관리가 생성되면 (報告管理が作成されたら)
7. 보고 관리가 업데이트되면 (報告管理が更新されたら)
8. 회사가 생성되면 (会社が作成されたら)
9. 상담이 생성되면 (商談が作成されたら)
10. 명사가 업데이트되면 (名刺が更新されたら)
11. 상담이 업데이트되면 (商談が更新されたら)
12. 리드가 생성되면 (リードが作成されたら)
13. 상담이 지정된 스테이지로 업데이트되면 (商談が指定のステージに更新されたら)

## 테스트 가능 여부
✅ 테스트 가능 (API 기반)

## 테스트를 위한 준비물

1. OAuth 자격증명 필요
2. Hot Profile 계정 (개발자 계정 권장)
3. API 엔드포인트 접근 권한
4. 테스트용 샘플 명사, 리드, 상담 데이터

## 사용 예시

```python
from integration import HotProfileClient

# 클라이언트 설정
client = HotProfileClient()

# 리드 검색
leads = await client.search_leads()

# 명사 정보 등록
await client.register_name_card(
    name="田中 太郎",
    email="tanaka@example.com",
    phone="090-1234-5678"
)

# 상담 등록
await client.register_opportunity(
    lead_id=123,
    opportunity_name="新規案件",
    stage="商談中"
)
```
