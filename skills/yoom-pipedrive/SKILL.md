# Pipedrive Yoom 연계 스킬

Pipedrive와 OpenClaw 연결을 위한 스킬입니다.

## 연계 정보
- **연계 방식**: API
- **인증 방식**: OAuth
- **API 액션**: 28개
- **트리거**: 10개

## 구현된 API 액션 (28개)
1. 조직 검색 (組織を検索)
2. 조직 삭제 (組織を削除)
3. 리드 업데이트 (リードを更新)
4. 조직 추가 (組織を追加)
5. 파일 추가 (ファイルを追加)
6. 거래 연결 상품 목록 (取引に紐づく製品一覧の取得)
7. 리드 검색 (リードを検索)
8. 활동 생성 (アクティビティを作成)
9. 인물 정보 업데이트 (人物情報を更新)
10. 리드 정보 가져오기 (リード情報を取得)
11. 조직 업데이트 (組織を更新)
12. 사용자 생성 (ユーザーを作成)
13. 거래 추가 (取引を追加)
14. 조직 연결 인물 가져오기 (組織に関連付けられた人物を取得)
15. 인물 추가 (人物を追加)
16. 활동 업데이트 (アクティビティを更新)
17. 조직 정보 가져오기 (組織情報を取得)
18. 거래 상세 가져오기 (取引の詳細を取得)
19. 인물 정보 검색 (人物情報を検索)
20. 인물 정보 가져오기 (人物情報を取得)
21. 리드 삭제 (リードを削除)
22. 노트 추가 (ノートを追加)
23. 인물 삭제 (人物を削除)
24. 활동 검색 (アクティビティを検索)
25. 거래 업데이트 (取引を更新)
26. 거래 삭제 (取引を削除)
27. 상품 목록 (製品一覧の取得)
28. 리드 생성 (リードを作成)

## 구현된 트리거 (10개)
1. 거래 삭제 (取引が削除されたら)
2. 인물 업데이트 (人物が更新されたら)
3. 조직 업데이트 (組織が更新されたら)
4. 거래 추가 (取引が追加されたら)
5. 거래 업데이트 (取引が更新されたら)
6. 인물 삭제 (人物が削除されたら)
7. 인물 추가 (人物が追加されたら)
8. 조직 삭제 (組織が削除されたら)
9. 조직 추가 (組織が追加されたら)
10. 활동 추가 (アクティビティが追加されたら)
