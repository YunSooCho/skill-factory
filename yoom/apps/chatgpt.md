## 🔄 연계 가능한 기능

### 1. 이메일 → ChatGPT 요약 → LINE 알림
특정 이메일 수신 시 ChatGPT가 본문 자동 요약 후 LINE 공식 계정 전송
- 외출 중 스마트폰으로 핵심 파악 가능
- 읽기 시간 및 중요 정보 누락 감소

### 2. 이메일 → ChatGPT 답안 → LINE 알림
수신 이메일 시점으로 ChatGPT가 답안 작성 후 LINE 공식 계정 알림
- 메일 점검 및 글 문면 작성 절차 감소
- 대처 시간 및 품질 편차 개선

### 3. 이메일 → ChatGPT 요약 → Slack
수신한 이메일 내용을 ChatGPT로 요약 후 지정 Slack 채널에 전송

### 4. 이메일 → ChatGPT 답안 → Slack 알림
수신한 이메일에 대한 답안을 ChatGPT로 작성 후 지정 Slack 채널 알림

### 5. 문의 → ChatGPT 자동 분류 → Slack 적절 채널
문의 이메일 수신 시 ChatGPT로 문의 내용 자동 분류 후 적절 Slack 채널 알림
- 문의 대처 속도 확보

### 6. Chatwork → ChatGPT 답변 → 알림
Chatwork에 게시한 내용에서 ChatGPT 답변 자동 갱신 후 지정 Chatwork 채널 전송
- 답변 내용, 전송 대상 채널, 메시지 사용자 정의 가능
