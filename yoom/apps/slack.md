## 🔄 플로 봇 오토메이션 (API 액션) - 21개

### API 액션 목록 (상위 10개):

1. **チャンネルにメッセージを送る** - 채널에 메시지 전송
2. **ダイレクトメッセージを送る** - 다이렉트 메시지 전송
3. **チャンネル内のメンバーID를取得する** - 채널 내 멤버 ID 가져오기
4. **チャンネル情報を取得** - 채널 정보 가져오기
5. **チャンネルを作成** - 채널 생성
6. **メッセージを削除** - 메시지 삭제
7. **ユーザー情報를取得** - 사용자 정보 가져오기
8. **ワークスペースの情報を取得** - 워크스페이스 정보 가져오기
9. **ファイルをアップロード** - 파일 업로드
10. **スタンプを追加** - 스탬프 추가

## 🎯 프로 봇 트리거 - 12개

### 트리거 목록 (상위 5개):

1. **メッセージがチャンネルに投稿されたら** - 메시지가 채널에 게시되면
2. **ファイルがチャンネルに投稿されたら** - 파일이 채널에 게시되면
3. **メッセージがチャンネルに投稿されたら（Webhook）** - 메시지가 채널에 게시되면 (Webhook)
4. **特定의キーワードを含むメッセージが投稿されたら** - 특정 키워드를 포함한 메시지가 게시되면
5. **スタンプが押されたら** - 스탬프가 눌리면

## 📋 템플릿 - 6개

### 템플릿 목록:

1. **Slackにメッセージが投稿されたら、別のチャンネルに転送する**
   - 내용: Slackの特定チャンネル投稿を自動で別チャンネルへ転送するワークフローです。Yoomで組み立てれば、手作業を減らし、共有漏れと転記ミスを防いで情報連携を安定させます。

2. **指定のユーザーを複数のSlackチャンネルに追加**
   - 내용: YoomデータベースにSlackチャンネルのリストを作成し、そのSlackチャンネルのリストへ指定のユーザーを自動的に追加します。‍

3. **Slackのメッセージにスタンプが押されたら、Google スプレッドシートにメッセージ内容を追加する**
   - 내용: Slackのメッセージにスタンプを押すだけで内容と投稿情報をGoogle スプレッドシートへ自動転記するフローです。手作業の入力やコピー＆ペーストを減らし、転記漏れや打ち間違いを防げます。

4. **Slackで特定のキーワードを含むメッセージを受信したら、Confluenceでページを作成する**
   - 내용: Slackの特定メッセージを検知しConfluenceへ自動でページを作成するフローです。転記の手間や抜け漏れを防ぎ、議事録や決定事項の蓄積を効率化します。

5. **Backlogの課題のコメントで自身がメンションされたらSlackに通知する**
   - 내용: Backlogの課題のコメントで自身がメンションされた場合のみ、Slackの特定チャネルに通知するフローボットです。通知先のチャンネルや通知内容を自由に変更してご利用ください。‍

6. **Slackのメッセージにスタンプが押されたら、Notionにメッセージ内容を追加する**
   - 내용: Slackの特定メッセージにスタンプを押すだけでYoomが内容をNotionデータベースへ自動転記するフローです。情報の見落としや転記ミスを抑え、チームのタスク管理と共有を効率化します。
