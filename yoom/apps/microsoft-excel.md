## 🔄 연계 가능한 기능

### 1. Outlook → OCR → Excel → Microsoft Teams 알림
Outlook 수신 메일을 OCR로 읽어 Microsoft Excel에 기록 후 Microsoft Teams로 공유
- 전서 절차 및 입력 오차 감소
- 문의/주문 대처 원활

### 2. 폼 → Microsoft Excel
입력 폼 전송 정보를 Microsoft Excel에 자동 저장
- Microsoft Excel 연동 웹 폼 간단 생성 가능

### 3. 신청 폼 → AI 분석 → Excel 견적서 → Outlook 발송
신청 폼 답변 전송 시, AI로 데이터 추출 후 Microsoft Excel 견적서 템플릿에서 견적서 작성, Outlook 메일 발송
- 수동 견적서 작성/발송 시간 대폭 단축
- 업무 효율화

### 4. 폼 답변 → Excel 견적서 → 메일 발송
폼에 답변된 내용을 사용하여 미리 준비한 엑셀 견적서 템플릿으로 견적서 발행 후 메일 발송
- 견적서 발행 및 발송을 폼 답변만으로 완료

### 5. 엑셀 템플릿 → 발주서/주문서 → 메일 발송
입력 폼 정보로 발주서/주문서 PDF 작성 후 메일 발송
- 발주서/주문서 템플릿은 Microsoft Excel 템플릿 사용

### 6. 이메일 → 키워드 추출 → Microsoft Excel
수신한 이메일 글문에서 특정 키워드 추출 후 Microsoft Excel에 자동 저장
- 추출 키워드 및 키워드를 저장하는 스프레드시트 자유 설정
- Gmail, Outlook 등에서 메일 전달 사용 가능
