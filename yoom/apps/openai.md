# OpenAI - Yoom 연계

## 🤖 기본 정보
- **서비스명:** OpenAI
- **URL:** https://lp.yoom.fun/apps/openai
- **카테고리:** 業務一般 (업무 일반)

---

## 🔄 연계 가능한 기능

### 1. 폼 → OpenAI 분석 → LINE 알림
폼 전송을 시점으로 OpenAI로 요약 후 LINE 공식 계정에 알림
- 확인 작업 절차 감소
- 공유 누락 방지
- 문의 대처 신속
- 업무 표준화

### 2. 폼 → OpenAI 분석 → Asana (태스크)
폼 답변을 시점으로 OpenAI가 내용을 분석/요약 후 Asana에 태스크 자동 추가
- 확인 → 태스크 절차 수동 절차 감소
- 대처 누락 및 입력 오차 방지
- 소요 시간 단축

### 3. 폼 → OpenAI → Zendesk (자동 답변)
폼 수신을 시점으로 특정 답변 경우 OpenAI가 답변 텍스트 생성 후 Zendesk에 자동 게시
- 초기 대응 원고 작성 절차 감소
- 담당자 균질 답변
- 시간 단축

### 4. 폼 → OpenAI 텍스트 생성 → LINE 알림
폼 전송을 시점으로 OpenAI가 답변 텍스트 생성 후 LINE 공식 계정에 자동 알림
- 정형문 작성/공유 수동 절차 절차 감소
- 대처 시간 단축
- 텍스트 품질 균일화

### 5. 폼 → OpenAI 요약 → Statuspage (인시던트)
폼 게시를 시점으로 OpenAI가 내용을 요약 후 Statuspage에 인시던트 자동 등록
- 보고 전서 절차 감소
- 오차 감소
- 장애 시 초기 확실한 정보 공유 및 속도 확보

### 6. 폼 → OpenAI 이미지 생성 → Google 스프레드시트
폼 전송을 시점으로 OpenAI로 이미지 생성 후 Google 스프레드시트에 기록
- 프롬프트 작성 및 데이터 정리 절차 감소
- 컨텐츠 제작 효율화
