## 🔄 플로 봇 오토메이션 (API 액션) - 13개

### API 액션 목록 (전체):

1. **グループの一覧を取得** - 그룹 목록 가져오기
2. **グループにメンバーを追加** - 그룹에 멤버 추가
3. **グループのメンバーの一覧を取得** - 그룹 멤버 목록 가져오기
4. **グループからメンバーを削除** - 그룹에서 멤버 삭제
5. **ユーザー情報를取得** - 사용자 정보 가져오기
6. **ユーザーを作成** - 사용자 생성
7. **ユーザーを更新** - 사용자 갱신
8. **ユーザーを削除** - 사용자 삭제
9. **組織単位情報を取得** - 조직 단위 정보 가져오기
10. **組織単位を作成** - 조직 단위 생성
11. **組織単位を更新** - 조직 단위 갱신
12. **組織単位を削除** - 조직 단위 삭제
13. **利用状況を確認** - 이용 현황 확인
