# Outlook - Yoom 연계

- **서비스명:** Outlook
- **URL:** https://lp.yoom.fun/apps/outlook
- **카테고리:** 業務一般 (업무 일반)

---

## 🔄 플로 봇 오토메이션 (API 액션) - 18개

### API 액션 목록 (상위 10개):

1. **自分のカレンダーにイベントを登録する** - 내 캘린더에 이벤트 등록
2. **予定表を作成する** - 캘린더 생성
3. **指定のユーザーのカレンダーにイベントを登録する** - 지정 사용자의 캘린더에 이벤트 등록
4. **イベントを作成** - 이벤트 생성
5. **イベントを更新** - 이벤트 갱신
6. **予定表の情報を取得** - 캘린더 정보 가져오기
7. **メールを送信** - 메일 발송
8. **メールの情報を取得** - 메일 정보 가져오기
9. **会議室の空き状況を確認** - 회의실 여부 확인
10. **会議室を予約** - 회의실 예약

---

## 🎯 프로 봇 트리거 - 6개

### 트리거 목록 (전체):

1. **特定の件名のメールを受信したら** - 특정 제목의 메일 수신 시
2. **メールを受信したら** - 메일 수신 시
3. **カレンダーにイベントが登録されたら** - 캘린더에 이벤트 등록 시
4. **会議室が予約されたら** - 회의실 예약 시
5. **会議室の予約がキャンセルされたら** - 회의실 예약 취소 시
6. **予定が変更されたら** - 예정 변경 시
