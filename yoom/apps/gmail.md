## 🔄 플로 봇 오토메이션 (API 액션) - 5개

### API 액션 목록:

1. **メールを検索** - 이메일 검색
2. **メッセージをゴミ коробкуへ移動する** - 메시지를 휴지통으로 이동
3. **特定のメッセージにラベルを追加** - 특정 메시지에 라벨 추가
4. **特定のメッセージのラベルを削除** - 특정 메시지의 라벨 삭제
5. **特定のメッセージを取得** - 특정 메시지 가져오기

## 🎯 프로 봇 트리거 - 3개

### 트리거 목록:

1. **特定のキーワードに一致するメールを受信したら** - 특정 키워드와 일치하는 이메일 수신 시
2. **特定のラベルのメールを受信したら** - 특정 라벨의 이메일 수신 시
3. **特定のラベルのメールを受信したら（ラベルID指定）** - 특정 라벨의 이메일 수신 시 (라벨 ID 지정)
