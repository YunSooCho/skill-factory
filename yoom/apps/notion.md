## 🔄 플로 봇 오토메이션 (API 액션) - 31개

### API 액션 목록 (상위 10개):

1. **ページ情報를取得** - 페이지 정보 가져오기
2. **ページを作成** - 페이지 생성
3. **ページにテキストを追加** - 페이지에 텍스트 추가
4. **特定のプロパティの値を取得** - 특정 프로퍼티 값 가져오기
5. **プロパティを更新** - 프로퍼티 갱신
6. **データベース의レコードを作成** - 데이터베이스 레코드 생성
7. **データベースのレコードを取得** - 데이터베이스 레코드 가져오기
8. **データベースのレコードを更新** - 데이터베이스 레코드 갱신
9. **ページを削除** - 페이지 삭제
10. **ブロックを追加** - 블록 추가

## 🎯 프로 봇 트리거 - 14개

### 트리거 목록 (상위 5개):

1. **ページが作成または更新されたら** - 페이지가 생성되거나 갱신되면
2. **データベースが作成または更新されたら** - 데이터베이스가 생성되거나 갱신되면
3. **【非推奨】特定のデータベースのページが作成・更新されたら** - 특정 데이터베이스의 페이지가 생성/갱신되면 (비추천)
4. **ページが削除されたら** - 페이지가 삭제되면
5. **特定のページが更新されたら** - 특정 페이지가 갱신되면
