- **カテゴリー:** 業務一般

---

## 🔄 連携可能な機能

### 1. メール(縦書きPDF) → AI → Googleドキュメント転記
メールで縦書きのPDF資料が届いたら、AIでテキストを抽出しGoogleドキュメントに内容を自動転記
- 手入力の手間をなくし、入力ミスを防止

### 2. フォーム → Googleドキュメント雇用契約書 → PDF発行
フォームの内容を用いて、Googleドキュメントの雇用契約書雛形からPDFを自動発行
- 雛形シートのコピー・入力・PDF発行の手動操作が不要

### 3. フォーム → Googleドキュメント雇用契約書 → PDF → Discord通知
フォーム内容からGoogleドキュメントの雇用契約書雛形でPDFを発行しDiscordに通知
- フォーム入力だけで書類発行が自動完了

### 4. 会議データ → 文字起こし → Googleドキュメント
入力フォームにアップロードした会議データから音声を文字起こしし、Googleドキュメントに自動作成
- Zoom、Meetなどのレコーディングデータに対応

### 5. メール添付書類 → AI読み取り → Googleドキュメント転記
メールで添付書類を受信したらAIで読み取りデジタル化し、自動でGoogleドキュメントに転記
- 手入力の手間を省きデータの入力ミスを防止

### 6. Googleフォーム → Googleドキュメント追加
Googleフォームの送信をトリガーに、回答内容をGoogleドキュメントの末尾へ自動追加
- コピー＆ペーストを省き、転記ミスを防止
