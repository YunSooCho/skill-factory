## 🔄 연계 가능한 기능

### 1. Zoom → Notion (화자 추록)
Zoom 미팅 종료 시 자동 화자 추록 및 Notion에 저장
- AI로 회화 내용을 자동으로 화자
- 기록을 Notion에 저장
- 자동 절차를 각 액션 설정 내용 변경 필요

### 2. Zoom → kintone (화자+요약)
Zoom 종료를 시점으로 녹음을 화자 추록 후 요약하며 kintone에 자동 등록
- 기록 작성 시간 감소
- 전서 오차 방지
- 회의 후 정보 공유 원활

### 3. Zoom → Microsoft Teams (화자+요약+알림)
Zoom 미팅 종료 시 자동으로 회화 내용을 화자 및 요약 Microsoft Teams에 알림
- AI 자동 절차 사용
- 회의 논의 및 결정사항을 정확하게 기록

### 4. Zoom → Google 스프레드시트 (녹음 정보)
Zoom 미팅 종료 시 레코딩 정보를 Google 스프레드시트로 추가
- Google 스프레드시트를 근간 데이타베이스로 정보 집약
- 데이터 공유 절차 감소

### 5. Zoom → Miro (새 보드)
기업 Zoom 미팅 종료 시 Miro에 새 보드 생성
- 기업용 보드를 생성
- 향후 영업 전략에 대해 팀 간 협의 원활

### 6. Zoom → LINE (화자+요약+알림)
Zoom 미팅 종료 시점으로 영상을 AI로 화자 후 요약 LINE 공식 계정으로 자동 알림
- 기록 작성 절차 감소
- 공유 누락 방지
- 회의 내용 즉시 커뮤니케이션
