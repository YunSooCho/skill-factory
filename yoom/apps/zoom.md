# Zoom - Yoom 연계

- **서비스명:** Zoom
- **URL:** https://lp.yoom.fun/apps/zoom
- **카테고리:** 業務一般 (업무 일반)

---

## 🔄 플로 봇 오토메이션 (API 액션) - 31개

### API 액션 목록 (상위 10개):

1. **ミーティングを作成（非推奨）** - 미팅 생성 (비추천)
2. **ミーティング情報を編集（非推奨）** - 미팅 정보 편집 (비추천)
3. **定期ミーティング（毎日）を作成（非推奨）** - 매일 정기 미팅 생성 (비추천)
4. **ミーティング情報を取得** - 미팅 정보 가져오기
5. **会議の一覧を取得** - 회의 목록 가져오기
6. ** Recording情報を取得** - 녹음 정보 가져오기
7. ** Recordingのダウンロードリンクを取得** - 녹음 다운로드 링크 가져오기
8. ** Recordingを削除** - 녹음 삭제
9. **チャットのメッセージを取得** - 채팅 메시지 가져오기
10. **チャットのファイルを取得** - 채팅 파일 가져오기

---

## 🎯 프로 봇 트리거 - 16개

### 트리거 목록 (상위 5개):

1. **ミーティングが作成されたら（Webhook）** - 미팅 생성 시 (Webhook)
2. **ミーティングが終了したら** - 미팅 종료 시
3. **会話が作成されたら** - 대화 생성 시
4. **参加者が変更されたら** - 참여자 변경 시
5. **Recordingが作成されたら** - 녹음 생성 시
